# Rock Paper Scissors Game 🪨 📄 ✂️

A simple, interactive terminal-based Rock-Paper-Scissors game built using Python. 

## 🚀 Features
- Play against an automated computer opponent.
- Score tracking system.
- Input validation (prevents wrong choices).

## 🛠️ How to Run
To play the game, you need to have Python installed on your system.

1. Clone or download the script (`game.py`).
2. Open your terminal/command prompt.
3. Run the following command:
   ```bash
   python game.py
